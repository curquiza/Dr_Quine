# Dr Quine

Introduction to quine : self-reproducing program.

3 versions:
- Colleen: print on stdout.
- Grace: print on a file.
- Sully : print in a new file, compile and execute this new file (in a loop from 5 to 0).

## In C language

```bash
$> cd C/
$> make
$> make test_all
```

## In Assembly

```bash
$> cd ASM/
$> make
$> make test_all
```
